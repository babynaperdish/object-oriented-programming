# object-oriented-programming
3rd semester
1) 1 семестр повторение.
2) Классы. Операции. Вложенность.
3) Шаблоны функций. Шаблоны классов. Стандартные шаблоны С++. Обработка исключений.
4) Контейнеры STL: deque, stack, queue, priority_queueset, multiset, map, multimap. Итераторы. Стандартные алгоритмы. Предикаты.
5) QT: Знакомство.
6) QT: Коллекция динамически добавляемых виджетов. Сигналы, слоты.
7) Qt: Реализация графического класса, содержащего коллекцию фигур, методов добавления новых фигур и удаления старых. Каждая фигура представлеяет из себя объект, который можно перемещать при помощи курсора, а также делать активным (объект поднимается на передний план), при наложении нескольких объектов друг на друга должен перемещаться только активный объект.
   
   7.1) Реализация с использованием пользовательской модели и рисования на виджетах через перегрузку методов: paintEvent, mouse(Press,Move,Release)Event.
   
   7.2) Реализация с использованием модели графического представления и сцены, границы фигуры строго контролируются.
